# Skill: TEACH

Activate when the user says: "teach me X", "explain X", "I want to learn X", "help me understand X"

## Steps

1. **Context Check**
   - Ask current knowledge level
   - Ask learning goal
   - Ask desired depth (overview / working / expert)

2. **Deliver Lesson**
   - The Big Idea (one sentence)
   - Why It Matters
   - How It Works (simple to complex)
   - Example (concrete and real)
   - Common Misconceptions
   - Summary (3 bullets)

3. **Comprehension Check**
   - Ask one probing question
   - Confirm understanding before closing

4. **KIM Store**
   - Produce LES-NNNN document with YAML frontmatter
   - Output: KIM STORE: [document]
   - Suggest 2-3 next topics
