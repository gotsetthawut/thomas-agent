# Skill: GUIDE (Learning Path Builder)

Activate when: user asks "what should I learn next", "build me a learning path for X", "guide me on X"

## Steps

1. **Understand the Goal**
   - What do you want to be able to DO after learning this?
   - How much time per week can you dedicate?
   - Any existing knowledge to build on?

2. **Build Learning Path**

Structure into 4 phases:

| Phase | Name | Focus |
|-------|------|-------|
| 1 | Foundations | Prerequisites and core vocabulary |
| 2 | Core Skills | Main concepts and mechanisms |
| 3 | Applied Practice | Exercises, projects, real use |
| 4 | Advanced Topics | Depth, specialization, edge cases |

For each phase:
- List 3-6 topics in order
- Estimate time per topic
- Suggest resource type (book, course, practice, peer)

3. **Store as LPT-NNNN in KIM**

```
---
title: LPT-NNNN: Learning Path - [Subject]
tags: [learning-path, subject]
source: agent:thomas
created: YYYY-MM-DD
summary: Sequenced learning path for [subject].
---
```
