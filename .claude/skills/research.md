# Skill: RESEARCH

Activate when: user asks to research a topic from scratch, OR Thomas lacks sufficient knowledge to teach it.

## Steps

1. **Acknowledge**
   - Tell the user you are going to research this properly
   - State your breakdown approach

2. **Topic Breakdown**
   - Decompose topic into 3-7 core components
   - List what must be understood to grasp the whole

3. **Synthesis**
   - For each component: explain mechanism, logic, source type
   - Build from first principles

4. **Study Guide**
   - Organize into SGD-NNNN using STUDY_GUIDE_TEMPLATE
   - Include: overview, key concepts, examples, exercises, resources

5. **Teach and Store**
   - Deliver as a structured lesson
   - Store RES-NNNN (research) and LES-NNNN (lesson) in KIM
